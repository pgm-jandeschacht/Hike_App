Charity of your choice

Walk-A-Wish has a bunch of different charities to chose from. With over 100 charities we have definitely something for you. In the app you can see the popular, new and most donated to charities. With this feature we help you find the prefect good cause match.

When you start a hike, mapped out or not, you will be able to select a good cause. By default this will be set to the last charity you have picked but you can always change this with the click or tap of a button.
